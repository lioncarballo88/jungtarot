# Jung Tarot

An Android app for tarot readings interpreted through the lens of Alejandro Jodorowsky's *La Vía del Tarot* and Carl Jung's analytical psychology. The app is not a fortune-telling tool — it uses tarot as a mirror for self-discovery and psychological reflection.

## Features

- **3 spread types**: La Cruz (3 cards), Cruz Celta (5 cards), La Herradura (7 cards)
- **AI-powered readings**: Narrative interpretations generated by Gemini 2.0 Flash, not static text
- **22 Major Arcana**: Complete Tarot de Marseille deck with Jodorowsky's restored Roman numerals (IIII, VIIII, XIIII, XVIIII)
- **Random or manual card selection**
- **Jungian archetypes**: Shadow, Anima, Self, Individuation, Synchronicity woven into every reading
- **Spanish UI**: Designed for Spanish-speaking users
- **Dark/light theme**: Follows system preference

## Getting Started

### Prerequisites

- Android Studio Hedgehog (2023.1.1) or later
- JDK 17
- Android SDK API 34
- A [Google AI Studio](https://aistudio.google.com) API key (free tier available)

### Setup

1. Clone the repository
   ```bash
   git clone https://github.com/lioncarballo88/jungtarot.git
   cd jungtarot
   ```

2. Add your Gemini API key to `local.properties`
   ```properties
   sdk.dir=/path/to/Android/sdk
   gemini.api.key=YOUR_API_KEY_HERE
   ```
   > `local.properties` is in `.gitignore` and will never be committed.

3. Open in Android Studio and wait for Gradle sync

4. Run on a device or emulator
   ```bash
   ./gradlew installDebug
   ```
